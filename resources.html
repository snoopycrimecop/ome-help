Other Resources

Omnigraffle® files for the creation of the screenshots used in the User Help website and other resources such as Word® files can be downloaded from the Help resources directory on the OME Downloads site:

  • Training Materials Resources

Pdfs are no longer being produced for each version but if you have a particular requirement for them for training purposes, please get in touch.

There are also movie tutorials on the OME YouTube Channel:

  • OME YouTube Channel

If you would like to build your own version of the User Help website, it is easily done using Jekyll, which is very quick to learn.

Simply clone the code from the ome-help GitHub repository and you are away:

  • ome-help GitHub repository

The Github Readme file explains the architecture of the website.
