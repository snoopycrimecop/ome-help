It is convenient to put together material for OMERO training sessions from some of the resources and PDFs available on the website.

We suggest using an A5 'Getting Started with OMERO' brochure which can be customised with your institution's server details. Generic UK and USA versions are in the downloads folder a5-brochure, as Word® Docs for editing, or as generic PDFs.

This generally helps the participants in the training session to get started and logged in with minimal confusion.

You can also select appropriate sections in the User Help website, and download the individual PDFs of the pages using the link at the top right of each page. These have been formatted for printing.

Alternatively all the PDFs available for both OMERO versions can be downloaded as a zip archive using the links below.

  • All User Guides for Version 5.1.1 (Zipped PDFs 144 MB)

  • All User Guides for Version 5.0.8 (Zipped PDFs 185 MB)

For users in the University of Dundee College of Life Sciences, there are some pages aimed specifically at users of Nightshade, the CLS OMERO server.
