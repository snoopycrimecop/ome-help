These pages contain the User Help for applications which can connect to an OMERO server or use OMERO plugins.

The User Guides for the current versions are listed in the main menu, for all OMERO applications, including OMERO.insight, OMERO.web, OMERO.editor and OMERO in ImageJ and Fiji.

The Guides for the current versions are listed in the main menu, and the Guides for Previous Versions page has links to archives of older versions.

The Resources page contains links to ZIP archives of the PDFs for all the current versions, customised versions of some of the Guides and other resources such as Omnigraffle® and Word® files.

The current release versions of OMERO.insight are 4.4.12 and 5.0.5. You can tell what version of OMERO.insight you are using by looking at the login screen.

New in OMERO.insight 5.0


Upgraded Search

The Search functionality in OMERO.insight and OMERO.web has been upgraded and harmonised between the two clients.

See the Search page for details of how to search for Images, Datasets, Projects, Screens, Plates and in annotations including tags, attachments and comments, plus the use of wildcards and other tips.





Using ImageJ with OMERO

The way that OMERO.insight-ij, the ImageJ plugin, uses Bio-Formats has changed. To use ImageJ with OMERO 5.0.x you now need bioformats_package.jar in the plugins folder, to replace loci_tools.jar, which must be removed.

Fiji now ships with Bio-Formats 5 by default so you can use it with the OMERO.insight-ij plugin version 5.0.x without needing to update anything.

Full details of how to use OMERO with ImageJ and Fiji for both versions 5.0.5 and 4.4.12, and how to upgrade, are shown in the Using ImageJ with OMERO section.
